## Periodicals

### Journal Article with DOI
```
Author, A. A., & Author, B. B. (Year). Title of article in sentence case.
    *Journal Name in Title Case, Volume*(Issue), Page-Page.
    https://doi.org/10.xxxx/xxxxx
```

### Journal Article without DOI (from database)
```
Author, A. A. (Year). Title of article. *Journal Name, Volume*(Issue), Page-Page.
```
Note: Omit database URL for common databases (JSTOR, EBSCO, ProQuest).

### Journal Article without DOI (from web)
```
Author, A. A. (Year). Title of article. *Journal Name, Volume*(Issue), Page-Page.
    https://direct-url-to-article
```

### Magazine Article (print)
```
Author, A. A. (Year, Month Day). Title of article. *Magazine Name, Volume*(Issue),
    Page-Page.
```

### Magazine Article (online)
```
Author, A. A. (Year, Month Day). Title of article. *Magazine Name*.
    https://url
```

### Newspaper Article (print)
```
Author, A. A. (Year, Month Day). Title of article. *Newspaper Name*, pp. Page-Page.
```

### Newspaper Article (online)
```
Author, A. A. (Year, Month Day). Title of article. *Newspaper Name*.
    https://url
```

### Special Issue
```
Editor, E. E. (Ed.). (Year). Title of issue [Special issue]. *Journal Name,
    Volume*(Issue).
```

## Books

### Book (print)
```
Author, A. A. (Year). *Title of book: Subtitle in sentence case*. Publisher.
```

### Book with DOI
```
Author, A. A. (Year). *Title of book*. Publisher. https://doi.org/10.xxxx/xxxxx
```

### Edited Book
```
Editor, E. E. (Ed.). (Year). *Title of book*. Publisher.
Editor, E. E., & Editor, F. F. (Eds.). (Year). *Title of book*. Publisher.
```

### Book Chapter in Edited Volume
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), *Title of book*
    (pp. xx-xx). Publisher. https://doi.org/10.xxxx/xxxxx
```

### Book with Edition
```
Author, A. A. (Year). *Title of book* (2nd ed.). Publisher.
```

### Book with Volume
```
Author, A. A. (Year). *Title of book* (Vol. 3). Publisher.
```

### Multivolume Work
```
Author, A. A. (Year-Year). *Title of work* (Vols. 1-4). Publisher.
```

### Republished Book
```
Author, A. A. (Year). *Title of book*. Publisher. (Original work published Year)
```
In-text: (Author, original year/republished year)

### Translated Book
```
Author, A. A. (Year). *Title of book* (T. Translator, Trans.). Publisher.
    (Original work published Year)
```
In-text: (Author, original year/translation year)

### Book in Non-English Language
```
Author, A. A. (Year). *Title in original language* [Translated title]. Publisher.
```

## Online Sources

### Webpage on a Website
```
Author, A. A. (Year, Month Day). Title of page. Site Name. https://url
```

### Webpage with Group Author
```
Organization Name. (Year, Month Day). Title of page. https://url
```
Note: Omit site name when same as author.

### Webpage with No Date
```
Author, A. A. (n.d.). Title of page. Site Name. https://url
```

### Webpage with No Author
```
Title of page. (Year, Month Day). Site Name. https://url
```
In-text: ("Title of Page," Year)

### Webpage Expected to Change (retrieval date needed)
```
Author, A. A. (n.d.). Title of page. Site Name. Retrieved Month Day, Year,
    from https://url
```

### Wikipedia Entry
```
Title of article. (Year, Month Day). In *Wikipedia*. https://archived-url
```
Use archived version URL. In-text: ("Title," Year)

### Online Dictionary/Encyclopedia Entry
```
Merriam-Webster. (n.d.). Word. In *Merriam-Webster.com dictionary*. Retrieved
    Month Day, Year, from https://url
```

### Blog Post
```
Author, A. A. (Year, Month Day). Title of post. *Blog Name*.
    https://url
```

### Online Forum/Reddit Post
```
Author [username]. (Year, Month Day). Title of post [Online forum post].
    Site Name. https://url
```

## Audiovisual Media

### Film/Movie
```
Director, D. D. (Director). (Year). *Title of film* [Film]. Production Company.
```

### Film from Streaming Service
```
Director, D. D. (Director). (Year). *Title of film* [Film]. Production Company.
    https://streaming-url
```

### TV Series
```
Executive Producer, E. P. (Executive Producer). (Year-Year). *Title of series*
    [TV series]. Production Company.
```

### TV Episode
```
Writer, W. W. (Writer), & Director, D. D. (Director). (Year, Month Day). Title
    of episode (Season X, Episode X) [TV series episode]. In E. P. Producer
    (Executive Producer), *Title of series*. Production Company.
```

### YouTube Video
```
Uploader Name [Username if different]. (Year, Month Day). *Title of video*
    [Video]. YouTube. https://www.youtube.com/watch?v=xxxxx
```

### TED Talk
```
Speaker, S. S. (Year, Month). *Title of talk* [Video]. TED Conferences.
    https://www.ted.com/talks/xxxxx
```

### Podcast (Entire Series)
```
Host, H. H. (Host). (Year-present). *Podcast title* [Audio podcast]. Producer.
    https://url
```

### Podcast Episode
```
Host, H. H. (Host). (Year, Month Day). Episode title (No. xx) [Audio podcast
    episode]. In *Podcast title*. Producer. https://url
```

### Webinar
```
Presenter, P. P. (Year, Month Day). *Title of webinar* [Webinar]. Organization.
    https://url
```

### Music Album
```
Artist or Group. (Year). *Title of album* [Album]. Record Label.
```

### Song/Track
```
Artist, A. A. (Year). Title of song [Song]. On *Title of album*. Record Label.
```

### Audiobook
```
Author, A. A. (Year). *Title of book* (N. Narrator, Narr.) [Audiobook].
    Publisher. https://url
```

## AI and Software

### AI Tool (ChatGPT, Claude, etc.)
```
Company Name. (Year). Tool Name (Version) [Large language model]. https://url
```

Examples:
```
OpenAI. (2024). ChatGPT (Mar 14 version) [Large language model].
    https://chat.openai.com/chat

Anthropic. (2024). Claude (3.5 Sonnet version) [Large language model].
    https://claude.ai

Google. (2024). Gemini [Large language model]. https://gemini.google.com
```

### Computer Software
```
Author, A. A. (Year). *Title of software* (Version X.X) [Computer software].
    Publisher. https://url
```

### Mobile App
```
Developer. (Year). App Name (Version X.X) [Mobile app]. App Store. https://url
```

### GitHub Repository
```
Author, A. A. (Year). *Repository name* (Version X.X) [Computer software].
    GitHub. https://github.com/username/repository
```

### Data Set
```
Author, A. A. (Year). *Title of data set* (Version X.X) [Data set]. Publisher.
    https://doi.org/xxxxx
```

## Legal and Government

### Court Case (U.S.)
```
Name v. Name, Volume Source Page (Court Year). URL
```
Example:
```
Brown v. Board of Education, 347 U.S. 483 (1954).
    https://www.oyez.org/cases/1940-1955/347us483
```
In-text: *Brown v. Board of Education* (1954)

### Federal Statute
```
Name of Act, Title Source ยง Section (Year). URL
```
Example:
```
Americans with Disabilities Act of 1990, 42 U.S.C. ยง 12101 et seq. (1990).
    https://www.ada.gov/pubs/adastatute08.htm
```

### Federal Regulation (Code of Federal Regulations)
```
Title/Number, Volume C.F.R. ยง Section (Year). URL
```

### Executive Order
```
Exec. Order No. XXXXX, 3 C.F.R. Page (Year). URL
```

### Government Report
```
Agency Name. (Year). *Title of report* (Report No. XXX). Parent Agency.
    https://url
```

### Congressional Hearing
```
Title of Hearing, xxx Cong. (Year). URL
```

### Patent
```
Inventor, A. A. (Year). Title of patent (U.S. Patent No. X,XXX,XXX).
    U.S. Patent and Trademark Office. https://url
```

### Technical Standard
```
Organization. (Year). *Title of standard* (Standard No. XXX). https://url
```

## Academic and Gray Literature

### Dissertation (Published, from database)
```
Author, A. A. (Year). *Title of dissertation* (Publication No. XXXXX)
    [Doctoral dissertation, University Name]. Database Name.
```

### Dissertation (Published, from repository)
```
Author, A. A. (Year). *Title of dissertation* [Doctoral dissertation,
    University Name]. Repository Name. https://url
```

### Dissertation (Unpublished)
```
Author, A. A. (Year). *Title of dissertation* [Unpublished doctoral
    dissertation]. University Name.
```

### Master's Thesis
Same format as dissertation, replace [Doctoral dissertation] with [Master's thesis].

### Conference Paper/Presentation
```
Presenter, A. A. (Year, Month Day-Day). *Title of presentation* [Paper
    presentation/Poster presentation/Conference session]. Conference Name,
    Location. https://doi.org/xxxxx
```

### Symposium Contribution
```
Contributor, A. A. (Year, Month Day-Day). Title of contribution. In C. Chair
    (Chair), *Title of symposium* [Symposium]. Conference Name, Location.
```

### Preprint
```
Author, A. A. (Year). *Title of manuscript*. PsyArXiv/bioRxiv/arXiv.
    https://doi.org/xxxxx
```

### Unpublished Manuscript
```
Author, A. A. (Year). *Title of manuscript* [Unpublished manuscript].
    Department Name, University Name.
```

### Manuscript in Preparation
```
Author, A. A. (Year). *Title of manuscript* [Manuscript in preparation].
    Department Name, University Name.
```

### Manuscript Submitted for Publication
```
Author, A. A. (Year). *Title of manuscript* [Manuscript submitted for
    publication]. Department Name, University Name.
```

### Working Paper/Technical Report
```
Author, A. A. (Year). *Title of report* (Working Paper No. XXX). Organization.
    https://url
```

### Press Release
```
Organization. (Year, Month Day). Title of press release [Press release].
    https://url
```

## Other Sources

### Personal Communication
In-text only (NOT in reference list):
```
(J. Smith, personal communication, January 15, 2023)
J. Smith (personal communication, January 15, 2023)
```

### Lecture Notes/Course Materials
```
Instructor, I. I. (Year). Title of lecture [Lecture notes/PowerPoint slides].
    LMS Platform. https://login-url
```

### Interview (Published)
Cite according to format (video, article, podcast).

### Interview (Unpublished)
Cite as personal communication (in-text only).

### Sacred/Religious Text
```
Version Title. (Year). Publisher. (Original work published Year)
```
Example:
```
King James Bible. (2017). King James Bible Online.
    https://www.kingjamesbibleonline.org/ (Original work published 1769)
```
In-text: (King James Bible, 1769/2017, 1 Cor. 13:1)

### Classical Work
Use standard numbering (act.scene.line, book.verse):
```
(Shakespeare, 1623/2003, 1.5.45-60)
(Homer, trans. 1990, 4.1-50)
```

### Diagnostic Manual (DSM, ICD)
```
American Psychiatric Association. (2022). *Diagnostic and statistical manual
    of mental disorders* (5th ed., text rev.).
    https://doi.org/10.1176/appi.books.9780890425787
```

### Clinical Guideline
```
Organization. (Year). *Title of guideline* (Guideline No. XXX). https://url
```

### Map
```
Cartographer or Organization. (Year). *Title of map* [Map]. Publisher.
```

### Infographic
```
Author, A. A. (Year). *Title of infographic* [Infographic]. Site Name.
    https://url
```

### Archived Webpage (Wayback Machine)
```
Author, A. A. (Year, Month Day). Title of page. Site Name.
    https://web.archive.org/web/YYYYMMDD/original-url
```
