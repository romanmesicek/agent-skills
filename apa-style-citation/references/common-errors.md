# APA 7th Edition: Common Errors Checklist

Use this checklist to identify and correct citation errors.

## In-Text Citation Errors

### 1. Ampersand vs. "And"

| Context | Correct | Incorrect |
|---------|---------|-----------|
| Parenthetical | (Smith & Jones, 2023) | (Smith and Jones, 2023) |
| Narrative | Smith and Jones (2023) | Smith & Jones (2023) |

### 2. Period Placement

| Correct | Incorrect |
|---------|-----------|
| ...end of sentence (Smith, 2023). | ...end of sentence. (Smith, 2023) |
| ...end of sentence (Smith, 2023, p. 45). | ...end of sentence (Smith, 2023, p. 45.) |

### 3. Et Al. Formatting

| Correct | Incorrect |
|---------|-----------|
| et al. | et. al. |
| et al. | et al |
| et al. | etal. |
| et al. | *et al.* (not italicized) |

**Usage:** Use from FIRST citation for 3+ authors (APA 7th change)

### 4. Page Number Formatting

| Correct | Incorrect |
|---------|-----------|
| p. 45 | p.45 |
| pp. 45-47 | pp. 45-7 |
| pp. 112-113 | pp. 112-13 |

**Rule:** List ALL digits in page ranges.

### 5. Missing Locators for Quotes

Direct quotations REQUIRE page, paragraph, section, or timestamp:
- (Smith, 2023, p. 45)
- (Smith, 2023, para. 4)
- (Smith, 2023, Methods section)
- (Smith, 2023, 2:15)

### 6. Year in Narrative Citations

| Context | Rule |
|---------|------|
| First mention in paragraph | Year required: Smith (2023) found... |
| Subsequent in same paragraph | Year optional: Smith also noted... |
| Parenthetical | Year always required: (Smith, 2023) |

### 7. Group Author Abbreviations

| Citation | Correct |
|----------|---------|
| First | (American Psychological Association [APA], 2019) |
| Subsequent | (APA, 2019) |

**Don't:** Abbreviate group author in reference list

### 8. Secondary Source Format

| Correct | Incorrect |
|---------|-----------|
| (Rabbitt, 1982, as cited in Lyon, 2014) | (Rabbitt, 1982, in Lyon, 2014) |
| | (Rabbitt, 1982, cited in Lyon, 2014) |

## Reference List Errors

### 9. DOI Format

| Correct | Incorrect |
|---------|-----------|
| https://doi.org/10.1037/0000185 | doi:10.1037/0000185 |
| | DOI: 10.1037/0000185 |
| | http://dx.doi.org/10.1037/0000185 |

### 10. Punctuation After DOI/URL

| Correct | Incorrect |
|---------|-----------|
| https://doi.org/10.1037/0000185 | https://doi.org/10.1037/0000185. |
| https://www.example.com | https://www.example.com. |

**Rule:** No period after DOI or URL

### 11. "Retrieved From"

| Context | Correct | Incorrect |
|---------|---------|-----------|
| Standard URL | https://url | Retrieved from https://url |
| With retrieval date | Retrieved January 15, 2026, from https://url | Retrieved from https://url on January 15, 2026 |

### 12. Title Capitalization

| Element | Case | Example |
|---------|------|---------|
| Article title | Sentence case | The effects of sleep on memory |
| Book title | Sentence case | *The psychology of learning* |
| Book subtitle | Sentence case after colon | *Learning to learn: New approaches* |
| Journal name | Title Case | *Journal of Educational Psychology* |
| Proper nouns in titles | Capitalize | Freudian theory in modern psychology |

### 13. Italicization Rules

| Italicize | Do NOT Italicize |
|-----------|------------------|
| Book titles | Article titles |
| Journal names | Chapter titles |
| Volume number | Issue number |
| Film/TV titles | Episode titles (unless standalone) |

**Example:** *Journal of Psychology, 45*(3), 112-128.

### 14. Issue Number Format

| Correct | Incorrect |
|---------|-----------|
| *45*(3) | *45(3)* |
| *45*(3) | *45*(*3*) |
| *12*, 45-67 (if no issue) | *12*(), 45-67 |

### 15. Author Name Format

| Correct | Incorrect |
|---------|-----------|
| Smith, J. A. | Smith, John A. |
| Smith, J. A., & Jones, B. | Smith, J. A. and Jones, B. |
| Smith, J. A., Jones, B., & Lee, C. | Smith, J. A., Jones, B. and Lee, C. |

### 16. Publisher Information

| Include | Exclude |
|---------|---------|
| Penguin Books | Penguin Publishing Co. |
| University Press | University Press, Inc. |
| Academic Press | Academic Press Ltd. |

**Omit:** Inc., Co., Ltd., Publishing, Publishers
**Keep:** Books, Press

### 17. Publisher Location

**APA 7th:** Publisher location NOT required (changed from 6th ed.)

| Correct | Incorrect |
|---------|-----------|
| Penguin Books. | New York, NY: Penguin Books. |

### 18. Edition and Volume

| Correct | Incorrect |
|---------|-----------|
| (2nd ed.) | (2nd Edition) |
| (Vol. 3) | (Volume 3) |
| (2nd ed., Vol. 3) | (2nd ed., volume 3) |

### 19. Database URLs

| Source Type | Include URL? |
|-------------|--------------|
| From JSTOR, EBSCO, ProQuest | No |
| From open web | Yes |
| ProQuest Dissertations | Yes (with database name) |
| Cochrane Database | Yes (with database name) |

### 20. Hanging Indent

| Correct | Incorrect |
|---------|-----------|
| First line flush left, subsequent indented 0.5" | All lines flush left |
| | First line indented |
| | Inconsistent indentation |

## Formatting Errors

### 21. References Heading

| Correct | Incorrect |
|---------|-----------|
| **References** (bold, centered) | References (not bold) |
| | REFERENCES |
| | Reference List |

### 22. Spacing

| Correct | Incorrect |
|---------|-----------|
| Double-space entire list | Single-spacing |
| No extra space between entries | Extra blank line between entries |

### 23. Alphabetization

| Rule | Example |
|------|---------|
| By first author surname | Adams before Brown |
| Ignore "A," "An," "The" in titles | "The Effects of..." filed under E |
| Nothing before something | Smith before Smith-Jones |
| Single author before multiple | Smith (2020) before Smith & Jones (2020) |

### 24. Correspondence Errors

**Every in-text citation must have a reference list entry.**
**Every reference list entry must have at least one in-text citation.**

Exceptions:
- Personal communications (in-text only)
- Entire source materials for meta-analyses (reference list note)

## Block Quotation Errors

### 25. Block Quote Formatting

| Element | Correct |
|---------|---------|
| Length trigger | 40+ words |
| Indentation | 0.5" from left margin (entire block) |
| Quotation marks | None |
| Spacing | Double-spaced |
| Citation placement | After final period |

**Correct:**
```
       The researcher concluded that the results were significant:
              The findings demonstrate a clear correlation between the
              variables, suggesting that further research is warranted in
              this area to determine the full extent of the relationship. (p. 45)
       The implications of these findings...
```

## Quick Error-Check Process

1. **Scan in-text citations:**
   - [ ] & in parenthetical, "and" in narrative
   - [ ] Period after citation
   - [ ] et al. for 3+ authors
   - [ ] Page numbers for all quotes

2. **Scan reference list:**
   - [ ] DOI format: https://doi.org/...
   - [ ] No period after DOI/URL
   - [ ] Sentence case for titles
   - [ ] Title Case for journal names
   - [ ] Only volume italicized, not issue

3. **Cross-check:**
   - [ ] Every citation has reference entry
   - [ ] Every reference has at least one citation
   - [ ] Author names match exactly

4. **Formatting:**
   - [ ] "References" bold and centered
   - [ ] Double-spaced throughout
   - [ ] Hanging indent on all entries
   - [ ] Alphabetized correctly
