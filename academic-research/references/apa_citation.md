# APA 7th Edition Citation Reference

## Inline Citations

| Authors | Format | Example |
|---------|--------|---------|
| 1 author | (Author, Year) | (Smith, 2023) |
| 2 authors | (Author & Author, Year) | (Smith & Jones, 2023) |
| 3+ authors | (First et al., Year) | (Smith et al., 2023) |
| Direct quote | (Author, Year, p. X) | (Smith, 2023, p. 45) |
| Multiple sources | (Author, Year; Author, Year) | (Jones, 2022; Smith, 2023) |
| No author | ("Short Title", Year) | ("Climate Report", 2023) |
| No date | (Author, n.d.) | (Smith, n.d.) |

## Reference List Formats

### Journal Article
```
Author, A. A., & Author, B. B. (Year). Title of article. *Journal Name, Volume*(Issue), Page–Page. https://doi.org/xxxxx
```

**Example:**
```
Smith, J. D., & Johnson, M. R. (2023). Effects of climate change on biodiversity. *Nature Climate Change, 13*(4), 234–245. https://doi.org/10.1038/s41558-023-01234-5
```

### Book
```
Author, A. A. (Year). *Title of work: Capital letter also for subtitle*. Publisher.
```

**Example:**
```
Anderson, K. L. (2022). *Climate science fundamentals: A comprehensive guide*. Oxford University Press.
```

### Edited Book Chapter
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), *Title of book* (pp. xx–xx). Publisher.
```

### Website
```
Author, A. A. (Year, Month Day). Title of page. Site Name. https://url
```

**Example:**
```
World Health Organization. (2023, March 15). Climate change and health. WHO. https://www.who.int/news-room/fact-sheets/detail/climate-change-and-health
```

### Report
```
Organization Name. (Year). *Title of report* (Report No. xxx). https://url
```

**Example:**
```
Intergovernmental Panel on Climate Change. (2023). *Climate change 2023: Synthesis report* (AR6 SYR). https://www.ipcc.ch/report/ar6/syr/
```

### Preprint
```
Author, A. A. (Year). Title of preprint. *Repository Name*. https://doi.org/xxxxx
```

**Example:**
```
Chen, L., & Park, S. (2024). Novel approaches to carbon capture. *arXiv*. https://arxiv.org/abs/2401.12345
```

### Dissertation/Thesis
```
Author, A. A. (Year). *Title of dissertation* [Doctoral dissertation, University Name]. Database Name.
```

## Special Cases

### DOI vs URL
- Prefer DOI when available: `https://doi.org/10.xxxx/xxxxx`
- Use URL only when no DOI exists
- Omit retrieval dates for stable content

### Author Name Formatting
- Last name, First initial. Middle initial.
- Up to 20 authors: list all
- 21+ authors: First 19, ..., Last author

### Title Capitalization
- Article/chapter titles: Sentence case
- Journal/book titles: Title Case (italicized)

### Missing Information
| Missing | Solution |
|---------|----------|
| Author | Use organization name or title |
| Date | Use (n.d.) |
| Title | Describe content in brackets: [Description] |
| Page numbers | Omit if not available |
